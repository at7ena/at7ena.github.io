Política de Privacidade

Esta política de privacidade descreve como o aplicativo da Aᴛ7ENA coleta, usa e protege as informações dos usuários.

1. Coleta de Informações

O aplicativo não coleta informações pessoais dos usuários. Todos os dados são utilizados apenas localmente no dispositivo.

2. Uso de Informações

As informações coletadas, caso existam, são utilizadas exclusivamente para melhorar a experiência do usuário e não são compartilhadas com terceiros.

3. Armazenamento de Dados

Os dados são armazenados de forma segura no próprio dispositivo. Não há envio de informações para servidores externos.

4. Alterações nesta Política

Esta política pode ser atualizada no futuro. Recomendamos que o usuário a revise periodicamente.

5. Contato

Em caso de dúvidas, entre em contato pelo e-mail: contato@at7ena.com
